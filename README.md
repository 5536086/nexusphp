完整的 PT 建站解决方案。基于 NexusPHP + Laravel Framewwork + Element Plus。

## 功能特性

- 发种
- 求种
- 字幕
- 考核
- 签到
- 论坛 
- 多语言
- 自动备份
- 管理后台  
- json api
- ....

## 系统要求
- PHP: 7.3|8.0，必须扩展：ctype, fileinfo, json, mbstring, openssl, pdo_mysql, tokenizer, xml, mysqli, gd
- Mysql: 5.7最新版或以上版本
- Redis：可选

更多信息请参考[文档](http://doc.nexusphp.org/)
