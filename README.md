完整的 PT 建站解决方案。基于 NexusPHP + Laravel Framework + Element Plus。

## 功能特性
- 发种
- 特别区  
- 求种
- 字幕
- 考核
- H&R
- 认领 
- 签到
- 补签卡  
- 勋章
- 自定义标签
- 论坛 
- 申诉  
- 多语言
- 自动备份
- 管理后台  
- Json API
- ....

## 系统要求
- PHP: 8.0，必须扩展：bcmath, ctype, fileinfo, json, mbstring, openssl, pdo_mysql, tokenizer, xml, mysqli, gd, redis, pcntl, sockets, posix
- Mysql: 5.7最新版或以上版本
- Redis：1.0.0或以上版本
